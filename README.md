# x86_asm 实验代码
这是阅读《x86汇编语言 从实模式到保护模式》一书时，每一章节的实验代码，包含详细注释。

本仓库适合：
- 正在学习这本书的读者参考
- 想了解 x86 实模式到保护模式切换全过程的爱好者学习
- 从零编写一个最小化操作系统内核入门学习

# 运行环境
## 依赖
需要安装以下工具：

```bash
# Ubuntu/Debian
sudo apt-get install nasm qemu-system-x86
```

- **nasm**：x86 汇编编译器
- **qemu-system-i386**：x86 模拟器，用来运行编译后的镜像

## 运行
进入每个章节文件夹，执行脚本运行：

```bash
cd cXX
sh run.sh
```

Qemu 会启动并显示运行结果。
